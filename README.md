💱 Conversor de Moedas em Python (Tkinter)

Um conversor de moedas simples desenvolvido em Python, utilizando Tkinter para a interface gráfica e consumo de API para obter valores atualizados.

 Funcionalidades:
 
Conversão entre diferentes moedas;
Interface gráfica simples e intuitiva;
Atualização de valores em tempo real (via API);
Fácil de usar.

 Tecnologias utilizadas:
 
Python;
Tkinter;
Requests (API de câmbio).

▶️ Como executar o projeto?

1. Clone o repositório
git clone https://github.com/fernandoflorenciodev/conversor-de-moedas-python-tkinter

3. Acesse a pasta
cd conversor-de-moedas-python-tkinter

4. Instale as dependências
pip install requests

5. Execute o programa
python nome_do_arquivo.py

 OBS:
É necessário ter o Python instalado;
Conexão com a internet é necessária para obter as taxas de câmbio atualizadas.

 Melhorias futuras:
