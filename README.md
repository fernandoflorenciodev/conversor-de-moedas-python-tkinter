# 💱 Conversor de Moedas em Python

Um conversor de moedas simples desenvolvido em **Python**, utilizando **Tkinter** para interface gráfica e consumo de API para obter valores atualizados.

---

##  Funcionalidades

* Conversão entre diferentes moedas
* Interface gráfica simples e intuitiva
* Atualização de valores em tempo real (via API)
* Fácil de usar

---

## 🛠️ Tecnologias utilizadas

* Python
* Tkinter
* Requests (API de câmbio)

---

## ▶️ Como executar o projeto

### 1. Clone o repositório

```
git clone https://github.com/fernandoflorenciodev/conversor-de-moedas-python-tkinter
```

### 2. Acesse a pasta

```
cd conversor-de-moedas-python-tkinter
```

### 3. Instale as dependências

```
pip install requests
```

### 4. Execute o programa

```
python nome_do_arquivo.py
```

---

##  Observações

* É necessário ter o Python instalado
* É necessário conexão com a internet para obter as taxas atualizadas
